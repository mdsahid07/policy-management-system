# 📘 Policy Hub – Policy Management System

A full-stack web application for creating, managing, and retrieving policies. Built with a modern technology stack, **Policy Hub** provides secure role-based access, powerful search/filtering, and a clean user experience for policy administrators and users.

---

## 🔗 Live Demo & Repository

## ⚙️ Tech Stack

### 🧩 Frontend
- React.js
- TypeScript
- Tailwind CSS (optional based on repo)

### 🔧 Backend
- Node.js
- TypeScript
- Express.js
- MongoDB (Mongoose ODM)

---

## 🔐 Features

- **Authentication & Authorization**
  - JWT-based login
  - Role-based access control (Admin/User)

- **Policy Management**
  - Create, edit, delete policies (Admin)
  - View and retrieve policies (All users)
  - Filter and search policies by title, category, date, etc.

- **RESTful API**
  - Well-structured endpoints for all operations
  - Follows HTTP best practices

- **User Roles**
  - **Admin**: Full control over all policies
  - **User**: View/search access only
